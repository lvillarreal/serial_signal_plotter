Graficar

Una vez realizada la medición, los datos capturados se encuentran almacenados en memoria RAM. Para graficar View → Graph → Graph data.

La gráfica es interactiva, por lo que puede realizar las siguientes acciones principales.

Zoom

  • Para crear una ventana de acercamiento, con click izquierdo presionado forme la ventana que desea visualizar. Para volver a zoom original presione click izquierdo y desplace hacia la izquierda.
  • Con click derecho ingresa a las opciones del gráfico, donde puede "acercar", "alejar" y seleccionar "escala automática" para ambos ejes.

ToolTips

  • Para determinar el valor de un punto en la gráfica (tanto para "x" como para "y") debe posicionar el ratón sobre el mismo y visualizar el tooltip.

Puntos de medición (shapes)

  • La gráfica competa los segmentos vacíos (entre muestra y muestra) con su unión mediante una recta. Para visualizar las muestras Config. → Graph → Set shapes visible.
