Guardar Como

Para guardar los datos de la medición, junto con las anotaciones realizadas, dirigirse a File → Save As... y seleccionar el destino donde desea guardar el archivo. Se generará un archivo con extensión .dat que posteriormente puede abrir en File → Open File...

AutoSave

Por seguridad, cuando finaliza la medición se realiza un guardado automático de los datos. Por lo tanto si desea recuperar la última medición, puede abrir el archivo de respaldo AutoSave.dat generado en el mismo directorio donde se encuentra el ejecutable de Signal Plotter.
