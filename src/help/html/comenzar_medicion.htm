Comenzar medición

Cuando presione el botón START comienza la captura de datos (el programa debe estar conectado con el dispositivo). En consola se muestra la información del dispositivo de medición.

Para finalizar la captura y poder visualizar la señal se debe presionar el botón DISCONNECT. En consola verá el tiempo de medición y un mensaje indicando que se ha desconectado el puerto correctamente. 

Es importante finalizar la captura de datos ya que Signal Plotter almacena el data-set en RAM, por lo que un tiempo de medida excecivo puede provocar un consumo innecesario de recursos del ordenador. Así mismo, se define un límite de seguridad para que esto no ocurra (ver apartado "Almacenamiento de la medición").
