Establecer conexión

La comunicación serial se realiza utilizando protocolo UART (Universal Asynchronous Receiver-Transmitter), por lo que es necesario que la velocidad de transmisión (baud rate) sea la misma que la del dispositivo de medición. 

Para setear el valor de baud rate debe dirigirse a Config. → Serial Comm → Set Baud Rate.

 

El valor por defecto es de 900000 baudios.

Luego, con el dispositivo conectado al puerto serial (puerto USB) debe identificar el puerto al que se ha conectado View → Serial Ports

Ingresar el puerto en el cuadro de texto (COM7 para este ejemplo) y presionar el botón CONNECT.

Si se ha conectado con el dispositivo verá el siguiente mensaje en consola
